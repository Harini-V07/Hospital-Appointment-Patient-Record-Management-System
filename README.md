# Hospital Appointment & Patient Record Management System

A web-based Hospital Management System developed using Python, Flask, and MySQL to streamline patient registration, doctor management, appointment scheduling, healthcare record maintenance, and report generation.

## Features

* Patient Management (Add, View, Update, Delete)
* Doctor Management
* Appointment Scheduling
* Patient Search Functionality
* Report Generation using File Handling
* Database Integration with MySQL
* User-friendly Web Interface using HTML, CSS, and Flask Templates

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* PyMySQL

## Python Concepts Implemented

* Object-Oriented Programming (OOP)
* CRUD Operations
* Exception Handling
* File Handling
* Lists, Tuples, and Dictionaries
* Modules and Packages
* Flask Routing and Templates

## Database Concepts Implemented

* SQL Joins
* Database Normalization
* Foreign Key Relationships
